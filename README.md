# Hello Bash
Make your unique bright and useful prompt in terminal! 🤓

![](/assets/hello-bash.gif)

# Quick installation 💾
```shell
eval "$(curl "https://raw.githubusercontent.com/michaelradionov/gg_installer/master/gg_installer.sh")" && gg_installer hello_bash
```

# Usage

Activation ✅
```sh
helloBash
```

Deactivation ❌
```sh
helloBash --clear
```
