# Hello Bash
Make your unique bright and useful prompt in terminal! 🤓 By the way you can use emoji (cmd+ctrl+space).

![](/assets/hello-bash.gif)

# Quick installation 💾
```sh
curl -L https://gogit.ru/hello | bash
```

# Usage

Activation ✅
```sh
hellobash
```

Deactivation ❌
```sh
hellobash --delete
```
